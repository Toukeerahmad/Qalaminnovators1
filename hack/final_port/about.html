This project, developed by Qalam Innovators, predicts potential disasters (earthquakes and floods) using real-time earthquake data from the USGS API and historical flood records. It leverages machine learning (Random Forest Classifier) to analyze past occurrences and forecast the most likely disaster and its severity for a given location.
